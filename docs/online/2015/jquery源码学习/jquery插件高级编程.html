promise 模式
promise代表某个操作的返回值
三个状态：unfulfilled，fullfilled，failed
一旦状态为fullfilled ，failed时promise的值就不会改变
$.Deferred 模式
$.when Deferred.done Deferred.fail
$.when 接受Deferred对象 把参数视为一个resolved的promise对象
没有参数直接触发done
接受不是Deferred对象 返回Deferred 的promise对象
Deferred.done 一个函数或者多个函数的数组 当promise对象成功resolved时触发
Deferred.fail  一个函数或者多个函数的数组 当promise对象成功rejected时触发
Deferred.then 是 done fail 合并体
Deferred.always 始终会执行的一个函数
Deferred.pipe 链接和过滤
接受三个参数 doneFilter 、failFilter、progressFilter
