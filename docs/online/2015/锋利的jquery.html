prototype

  • 内置对象做了大量的扩充
  • 代码比较独立，可直接使用
  • 面向对象不够强大，结构松散

dojo

  • 离线存储的api
  • 生成图标的组件
  • 基于SVG/VML的矢量图形库
  • 面向对象不够强大，结构松散

YUI (user interface library)

  • DOM操作
  • ajax应用

GWT (google web toolkit)

  • 通过GWT函数将java编译成javascript代码

Ext js

  • 界面扩展库来使用
  • 部分收费

moo tools

  • 扩展和兼容性

jquery

  • 强大的选择器
  • 出色的DOM操作
  • 可靠的事件处理
  • 出色的兼容性
  • 链式操作
