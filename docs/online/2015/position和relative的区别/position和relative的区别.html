position

标准

absolute
1、脱离文档流，一种布局方式
2、left和top相对父类是postion或者是relative
3、默认情况下left和top是相对于body。
会改变inline元素为inline-block
5、如果没有设置left的和top值的话，这个position属性就相当于relative,唯一不同的是它仍然脱离文档
6、bottom和right优先级比top和left小。所以设置了left，那么right就会失效
7、bottom和right设置为0，那么元素距离能见度最下边际距离是滚动的高度
8、同z-index同级元素后者层次高于前者
9、同级元素z-index越高，层次越高
10、默认z-index为0
11、同级父类未设置z-index情况下，子类设置了z-index，那么这个z-index的值传递到父类
12、js在不设置left和top也是可以获取position的值为absolute
relative
1、不会脱离文档流，一种布局方式
2、left和top相对父类是postion或者是relative或者兄弟定位,而且设置了left和top不会影响结构
