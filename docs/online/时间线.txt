
1991 年正式发布了。WorldWideWeb 比较简单，
只有“返回”、“上一步”、“下一步”三个按钮，
但 WorldWideWeb 同时还是一个网页编辑器，具有网页编辑功能

1992
也加入了“历史记录”功能，有“前进”、“后退”图标，点击链接后文字颜色会发生改变
1993 年
已经支持 Html3 表单、表格等元素了，也是当时主流的浏览器。不过速度奇慢
Arena 是第一个支持背景图片、表格等的浏览器

1994-1995年
是全球第一款 Windows 平台上的浏览器
不仅支持 HTML 3，还将邮件和新闻进行了整合，而且用户还能很方便地查看历史记录
2.x 版支持了大部分 HTML 3 元素，并且开始支持 Javascript

1996 年开始的 3.x 界面仍然没什么太多的改变，但是增加了很多的插件，同时支持了有背景颜色的表格。
邮件管理、阅读器和一个即见即所得的网页编辑器

ie内核发展史
1995发布IE1.0
1997 年，IE4 伴随着 Windows 98 一起发布
2001年10月25日 window xp发布，IE6 捆绑在 Windows XP 中，其实，在 Windows XP 没有发布之前，IE6 就发布了
IE4 也是 IE 浏览器中第一个支持 DHTML 的版本。
2006 年 10 月推出了 IE7，
2009年3月19日发布ie8
2011 年发布了 IE9。
2012年10月最新版本的 ie10
2013年11月07日随Windows 8.1发行 ie11
